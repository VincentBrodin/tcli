# tcli
---
Typing test without leaving the terminal

## Usage

Default settings

```bash
tcli
```

Custom word length `-l`

```bash
tcli -l 20
```

## Installation

### go
```bash
go install github.com/VincentBrodin/tcli@latest
tcli
```

### source
```bash
git clone https://github.com/VincentBrodin/tcli
cd tcli
go build -o tcli
./tcli
```
